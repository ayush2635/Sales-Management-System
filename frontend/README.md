# Trustate Assignment - Frontend

This is the frontend application for the Retail Sales Management System, built with **Next.js**.

## Deployment

### Prerequisites

- Node.js (v18+)
- Backend server running (see backend README)

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

### Building for Production

```bash
npm run build
npm start
```
